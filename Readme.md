# Исследование галлюцинаций ASR-моделей (Whisper)

Проект посвящен анализу и решению проблемы генерации ложного текста («галлюцинаций») нейросетями семейства Whisper при обработке музыки и фонового шума.

## Описание проекта

В ходе работы было проведено исследование того, как модели разного масштаба (от `tiny` до `large-v3`) реагируют на отсутствие речевого сигнала. Было доказано, что простое увеличение количества параметров модели не избавляет от артефактов, а в некоторых случаях (на примере классической музыки) делает их более сложными и непредсказуемыми.

**Основные этапы работы:**
* **Визуальный анализ**: Построение спектрограмм аудиосигналов для сопоставления акустических характеристик и поведения моделей.
* **Стресс-тестирование**: Прогон моделей на треках с разной амплитудой и жанрами (Classical, Gaming Ambient, Lo-Fi, Stock Comedy).
* **Архитектурное решение**: Разработка и бенчмаркинг гибридного пайплайна **Silero VAD + Whisper Small**, который отсекает большинство неречевых фреймов.

Теоретические выводы в проекте подкреплены ссылками на профильную литературу (И. Гудфеллоу, «Глубокое обучение») и документацию OpenAI.

## Стек технологий

* **Python 3.9+**
* **faster-whisper**: Оптимизированный движок для транскрибации.
* **Silero VAD**: Нейросетевой фильтр голосовой активности.
* **Librosa / Matplotlib**: Инструменты для визуализации аудио.
* **Pandas**: Аналитическая обработка результатов бенчмарка.
* **PyTorch**: Инференс VAD-модели.

## Структура проекта

- `main.ipynb` — основное исследование (кодировка UTF-8).
- `requirements.txt` — зависимости для CPU-окружения.
- `.gitignore` — конфиг исключений Git.
- `songs/` — папка для аудиофайлов (необходимо самостоятельно распоковать songs.zip в папку проекта).
